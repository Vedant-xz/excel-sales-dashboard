# excel-sales-dashboard
Interactive Excel sales dashboard using slicers and VBA macros.

## Features
- City-wise data filtering using slicers
- Multiple dashboards for sales and target analysis
- VBA macros with checkboxes to show or hide dashboards
- Pivot tables and charts

## Tools Used
- Microsoft Excel
- Pivot Tables & Pivot Charts
- Slicers
- VBA (Macros)

## Notes
This project was created for learning purposes by following online tutorials.
